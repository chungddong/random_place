# Random Place

모바일컴퓨팅 과목 과제 - 랜덤 장소 추천 앱

## 프로젝트 개요

Flutter와 카카오맵 API를 활용하여 랜덤으로 장소를 추천해주는 모바일 애플리케이션입니다.

## 주요 기능

- 랜덤 장소 추천
- 카카오맵 API 연동
- 장소 정보 표시

## 개발 환경

- Flutter 3.32.8
- Dart 3.8.1
- 지원 플랫폼: Android, Web

## 실행 방법

```bash
# 의존성 설치
flutter pub get

# Android 실행
flutter run

# Chrome 실행
flutter run -d chrome
```
